# Virtusa Technical Assessment - Mini Projects

This repository contains a suite of technical solutions developed for the Virtusa Performance Evaluation. Each project is designed to solve a specific business use case using industry-standard coding practices.

## 📁 Repository Structure

* **/SQL_DigitalLibrary**: A relational database audit system for library management. 
* **/Java_FinSafe_Validator**: A digital wallet transaction processor with robust error handling. 
* **/Python_OpsBot**: An automated security log analyzer for IT operations.

## 🛠 Tech Stack
* **SQL**: SQLite / MySQL for Relational Data Management.
* **Java**: Core Java (OOP, Exceptions, Collections).
* **Python**: Scripting, File I/O, and System Automation.
